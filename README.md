# Show Hide Password Project

This is my first React project created while learning React.js.

The project allows users to:
- Show password
- Hide password
- Toggle password visibility easily

## Technologies Used
- React.js
- Vite
- JavaScript
- CSS

- Conditional rendering
- Basic UI designing
